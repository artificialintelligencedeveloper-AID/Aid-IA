## Comprehensive Prompt System for Text-Only Commercial AI Agent

### Overview
The AIDA Elite V10 is designed to streamline interactions with clients through an optimized workflow, ensuring that every communication is efficient and purpose-driven. This document outlines the comprehensive prompt system that serves as the backbone of the AI agent's capabilities.

### Purpose of the AI Agent
- To facilitate seamless communication between businesses and clients.
- To enhance customer satisfaction through timely and accurate responses.
- To streamline operational efficiency by reducing manual input required from employees.

### Key Features
- **Natural Language Processing (NLP)**: Understands client inquiries and provides relevant responses.
- **Multilingual Support**: Capable of communicating in multiple languages.
- **Learning Capabilities**: Improves responses over time based on previous interactions.

### System Requirements
- n8n workflow for integration
- API access for third-party applications

### Prompt Structure
1. **Greeting**
   - "Hello! How can I assist you today?"
   - "Hi there! What brings you here today?"

2. **Information Gathering**
   - Prompt client for their needs: "Can you please describe your issue in detail?"
   - Collect specific data: "What service are you interested in?"

3. **Clarification**
   - If needed, ask follow-up questions: "Could you provide more details about that?"

4. **Response Generation**
   - Based on the gathered information, generate a tailored response.

5. **Closing**
   - Thank the client: "Thank you for reaching out! If you have any other questions, feel free to ask."

### Workflow Integration with n8n
- Use triggers for the initial inquiry.
- Set conditions to identify the type of service requested.
- Utilize function nodes to format responses based on user input.

### Optimization Techniques
- Regularly update prompt libraries based on user feedback.
- Automate repetitive responses to common inquiries.

### Testing and Feedback
- Implement A/B testing for different prompt structures.
- Gather feedback from both users and clients to improve interaction.

### Conclusion
This prompt system is designed to provide a framework for creating effective and efficient communications for the AIDA Elite V10 text-only commercial AI agent. Continuous refinement and implementation of user suggestions will ensure it remains optimized for various workflows, especially through n8n integration. Stay tuned for updates as the AI agent evolves!
